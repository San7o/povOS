# Resources

- [MultiProcessor
Specification](https://web.archive.org/web/20121002210153/http://download.intel.com/design/archives/processors/pro/docs/24201606.pdf) (legacy, replaced by ACPI)
- [ACPI](https://uefi.org/htmlspecs/ACPI_Spec_6_4_html/): standard for
  "Plug and Play" devices and their resources. The specification
  provides the structure of all types of tables.
